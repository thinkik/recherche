# Recherche-Cockpit

Das Recherche-Cockpit ist ein interaktives Board, um Recherche-Tools und Notizen entlang von sechs Phasen des wissenschaftlichen Arbeitens zu organisieren. Das Board ist als einzelne HTML-Datei umgesetzt und läuft komplett im Browser.

## Features

- Drag-&-Drop-Hangar mit Werkzeugen, Kategorien und Badges (KI/Uni-Lizenz).
- Sechs Recherche-Phasen mit Hinweisen und Platz für Notizen.
- Guided Tour & Onboarding inklusive KI- und Quellen-Checkliste.
- Eigene Tools hinzufügen sowie lizenzierte Datenbanken ergänzen.
- Export/Import als JSON und PDF-Ausgabe via Druckdialog.
- Lokale Speicherung des Cockpit-Status im Browser (LocalStorage).

## Verwendung

1. Repository klonen.
2. `index.html` im Browser öffnen.
3. Tools aus dem Hangar in die Phasen ziehen, Notizen hinzufügen und das Cockpit konfigurieren.

## Daten & Persistenz

Das Cockpit speichert den aktuellen Stand, benutzerdefinierte Tools sowie gelöschte Einträge lokal im Browser (LocalStorage). Zum Teilen oder Sichern steht der JSON-Export zur Verfügung.
